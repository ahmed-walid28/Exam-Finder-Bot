# 🎓 Exam Information Bot | بوت معلومات الامتحانات

بوت تيليجرام ذكي للطلاب للحصول على معلومات الامتحانات (المادة، مكان الامتحان، الوقت)

## المميزات ✨

- ✅ اختيار المادة من قائمة تفاعلية
- ✅ البحث السريع برمز الطالب (Student ID)
- ✅ عرض معلومات الامتحان كاملة
- ✅ دعم اللغة العربية الكامل
- ✅ واجهة سهلة وودية

## التثبيت 📦

### 1. تثبيت Python (إذا لم يكن مثبتاً)
- اذهب إلى [python.org](https://www.python.org)
- حمّل Python 3.9 أو أحدث

### 2. تثبيت المكتبات المطلوبة
```bash
pip install -r requirements.txt
```

### 3. الحصول على بوت توكن من Telegram
1. افتح Telegram وابحث عن `@BotFather`
2. اكتب `/newbot` واتبع التعليمات
3. ستحصل على توكن (رمز طويل)

### 4. تعديل البوت توكن
افتح ملف `bot.py` وجد هذا السطر:
```python
application = Application.builder().token("YOUR_BOT_TOKEN_HERE").build()
```

استبدل `YOUR_BOT_TOKEN_HERE` بالتوكن الذي حصلت عليه من BotFather

## طريقة الاستخدام 🚀

### تشغيل البوت
```bash
python bot.py
```

### خطوات الاستخدام في Telegram
1. ابحث عن البوت الخاص بك في Telegram
2. اكتب `/start`
3. اختر المادة من القائمة
4. أدخل كود الطالب (الـ ID)
5. ستحصل على معلومات الامتحان فوراً ✅

## النشر على Railway 🚆

المشروع جاهز للنشر على Railway كخدمة خلفية، فقط اربط المستودع ثم عرّف متغير البيئة `BOT_TOKEN`.

### الخطوات السريعة
1. أنشئ مشروع جديد في Railway.
2. اختر ربط المستودع الحالي من GitHub.
3. أضف متغير البيئة `BOT_TOKEN` بقيمة توكن البوت.
4. اترك Railway يقرأ ملف `Procfile` ويشغّل `python bot.py` تلقائياً.
5. نفّذ النشر وانتظر حتى تظهر حالة الخدمة كـ Running.

### ملاحظات
- لا تحتاج إلى منفذ `PORT` لأن البوت يعمل بنمط polling.
- تأكد أن مجلد `Data/` موجود داخل المستودع حتى يتم تحميل ملفات الامتحانات.

## الأوامر المتاحة 🎮

- `/start` - بدء البوت واختيار المادة
- `/help` - عرض تعليمات الاستخدام

## استكشاف الأخطاء 🔧

### الخطأ: "لا توجد بيانات متاحة"
- تأكد من وجود ملفات .txt في مجلد `Data`
- تأكد من تنسيق البيانات (CSV مع رؤوس)

### الخطأ: "لم يتم تحميل البوت"
- تأكد من تثبيت المكتبات: `pip install -r requirements.txt`
- تأكد من صحة البوت توكن

### الخطأ: "لم يتم العثور على الكود"
- تأكد من كتابة الكود الصحيح
- الكود يجب أن يكون موجوداً في ملف المادة المختارة

## نصائح 💡

1. يمكنك البحث عن عدة مواد بالتتابع
2. البوت يدعم البحث بدقة - تأكد من الكود
3. البيانات تُحدّث تلقائياً عند إضافة ملفات جديدة
